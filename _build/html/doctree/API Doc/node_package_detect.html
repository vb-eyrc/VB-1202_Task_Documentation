node_package_detect¶

This script handles the detection of the colour of the package under the camera.

class node_package_detect.Camera1[source]¶

Camera class to detect on-shelf packages.

Variables:
  • counter (int) – Counts the number of packages detected.
  • package_colour (list(str)) – Lists all the packages detected.
  • frame – Frame for object detection.
  • bridge – OpenCV bridge variable.
  • image_sub (rospy.Subscriber) – Subscriber for the shelf camera image data.
