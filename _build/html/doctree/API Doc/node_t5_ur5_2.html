node_t5_ur5_2¶

Controls the UR5 near the bins to pick up boxes from the conveyor. Borrows methods from lib_task5 to aid execution, and uses multithreading to operate the UR5_2 and conveyor simultaneously.

class node_t5_ur5_2.UR5Two[source]¶

Class for the module.

Variables:
  • package_pos (float) – Stores the conveyor position of the detected box.
  • package_name (str) – Stores models’ names from camera_callback().
  • package_colours (list(str)) – Stores package colours from package_callback().
  • ready_flag (bool) – Start-stop variable for operation of the UR5_2.
  • order (dict) – The current order to be shipped.
  • sub_package_colour (rospy.Subscriber) – Subscriber to receive the list of colours from the Logical Camera.
