node_t5_ur5_1¶

This script controls the UR5 near the shelf for placing boxes on the conveyor. It borrows methods from lib_task5 for execution, and uses multithreading to operate controller and the order pipeline simultaneously.

  • hp_list (list(dict)) – Queue of High-Priority packages.

  • lp_list (list(dict)) – Queue of Low Priority packages.
